IoT in Modern Farming: Smart Agriculture Project

Presented at: High School Science Festival, 2024

Project Overview

For a high school science festival, I developed a smart agriculture monitoring system using an ESP32-CAM, Blynk IoT platform, and a SIM900A GSM module. The project demonstrates how IoT can automate and monitor key environmental factors in farming, making agriculture smarter and more efficient.

Features

  • Monitors soil moisture, pH, rainfall, light, temperature, and humidity
  • Displays real-time data on an LCD
  • Remote monitoring via Blynk mobile app
  • Automatic SMS alerts and sensor data on incoming calls
  • Automated water pump control based on soil moisture

Hardware Used

  • ESP32-CAM microcontroller
  • SIM900A GSM module
  • DHT11 temperature & humidity sensor
  • Soil moisture, pH, rain, and light sensors
  • Relay module for pump control
  • LiquidCrystal I2C LCD display

How It Works

  1. ESP32 reads sensor data and displays it on the LCD.
  2. Sensor values are sent to the Blynk app for remote monitoring.
  3. On receiving a call, the system sends current sensor data via SMS using SIM900A.
  4. If soil moisture drops below 40%, the relay turns on the water pump automatically.
